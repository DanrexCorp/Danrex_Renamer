 # Danrex Renamer

Утилита для массового переименования файлов. Входит в состав **Danrex Tools** — софт-направления **Danrex Corp**.

---

## Возможности

- Добавить префикс (например, `vacation_`)
- Добавить суффикс (например, `_final`)
- Заменить текст в названиях
- Пронумеровать файлы (`001`, `002`, `003`)
- Удалить символы по позиции
- Предпросмотр перед применением
- Не требует установки

---

## Режимы переименования

| Режим | Пример ДО | Пример ПОСЛЕ |
|-------|-----------|--------------|
| Префикс "photo_" | `image.png` | `photo_image.png` |
| Суффикс "_done" | `report.pdf` | `report_done.pdf` |
| Замена "old" → "new" | `old_file.txt` | `new_file.txt` |
| Нумерация (001) | `file.txt`, `doc.pdf` | `001.txt`, `002.pdf` |
| Удалить символы 1-3 | `abc_name.txt` | `_name.txt` |

---

## Как использовать

1. Скачайте **Danrex Renamer.exe**
2. Запустите (двойной клик)
3. Выберите папку
4. Выберите режим переименования
5. Введите значение
6. Нажмите «Предпросмотр»
7. Нажмите «Переименовать»

---
